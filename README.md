Архивная версия веб-карты, используемой Тау Китой в 2014-2017 годах.

Содержит новогодние коллажи - склейку карты из скриншотов игроков за прошедший год.

Для запуска локально:

1. Скачать
2. Распаковать архивы в ``./tiles-layers``
3. Открыть ``index.html`` в браузере

Родственный репозиторий с архивными картами: https://github.com/TauCetiStation/maps-archive/
